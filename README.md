# Mercari 検索フィルター

メルカリの検索を自動で **「新しい順・個人出品」** に設定するChrome拡張機能です。

## インストール手順

> GitHubのアカウントは不要です。

### 1. ファイルをダウンロード

右上の緑の「Code」ボタン →「Download ZIP」でダウンロードし、解凍する。

### 2. Chromeに読み込む

1. Chromeのアドレスバーに `chrome://extensions/` と入力して開く
2. 右上の「**デベロッパーモード**」をONにする
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
4. 解凍したフォルダ（`mercari-filter`）を選択する

### 3. 完了

メルカリで検索すると、自動で「新しい順・個人出品」が適用されます。
