# Mercari 検索フィルター v2

メルカリの検索に自動でフィルターを適用するChrome拡張機能です。
ポップアップから各フィルターを個別に ON/OFF できます。

## 機能

ポップアップ上部の「すべて有効」スイッチで全フィルターを一括 ON/OFF できます。

| トグル | 内容 | デフォルト |
|--------|------|-----------|
| 新しい順 | 出品日時の降順でソート | ON |
| 個人出品のみ | ショップ出品を除外 | ON |
| 売切を表示 | 売り切れ商品も含める | ON |
| 最低価格フィルター | 指定金額以上に絞り込み（直接入力可） | ON（3,000円） |
| 最高価格フィルター | 指定金額以下に絞り込み（直接入力可） | OFF（10,000円） |
| フレーズ検索 | 検索語を `" "` で囲んでフレーズ検索 | ON |

拡張機能アイコンをクリックして各フィルターを切り替え可能。

## インストール手順

> GitHubのアカウントは不要です。

### 1. ファイルをダウンロード

右上の緑の「Code」ボタン →「Download ZIP」でダウンロードし、解凍する。

### 2. Chromeに読み込む

1. Chromeのアドレスバーに `chrome://extensions/` と入力して開く
2. 右上の「**デベロッパーモード**」をONにする
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
4. 解凍したフォルダ（`mercari-filter2`）を選択する

### 3. 完了

メルカリで検索すると、自動でフィルターが適用されます。
ブラウザ右上の拡張機能アイコンから各フィルターの ON/OFF が可能です。
